Installation
1

Drag the button to your bookmarks bar

2

How to use

1
Open mattw.io Navigate to mattw.io/youtube-metadata/bulk
2
Load the data Paste a channel, playlist, or video URL → click Submit → wait for the full table to load
3
Click the bookmarklet Click MW Bulk Export in your bookmarks bar — an overlay with a preview will appear
4
Choose format and download CSV, TSV, JSON or Markdown — the file downloads instantly with a timestamp in the filename
✕
Close the overlay Click × or run the bookmarklet again — the overlay will close
